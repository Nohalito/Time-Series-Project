# Time-Series-Project
This is a group project of our Econometric &amp; Data Science Master, more precisely, our **Time Series** class.

## Repository structure :

- **Final_Report.rmd** :
Will contain all of our answers and code of the 5 exercises.
- **datasets/ folder** :
Contain all the needed dataset to answer this project question.
- **Ex_i/ folder** :
For each of the 5 given exercise, contain the following :
    
    - **Ex_i.rmd** :
    Contain all the answers and code to solve the i-th exercise.
    - **Ex_i.pdf** :
    Pdf generated after knitting the associated R markdown (.rmd). 
    - **guideline.txt** :
    Contain the instruction for the associated exercise in a txt format.
